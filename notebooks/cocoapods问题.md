# CocoaPods问题及解决

## 命令

查看当前版本号

```
pod --version
```

更新cocoapods

```
sudo gem install cocoapods
```

卸载cocoapods

```
sudo gem uninstall cocoapods
```

更新gem

```
sudo gem update --system
```

## 问题

```
None of your spec sources contain a spec satisfying the dependency: `RxSwift (~> 4.0)`.
```

执行命令

```
pod setup
```
